Teleporterは2人対戦用TPSゲームで、プレイヤーは「テレポート」は駆使して戦います。
<br>windows10のみプレイが可能です。
まだ、開発途中のため、オンライン上に3人以上いるときプレイに支障きたす場合がございます。

# 操作方法
## ホーム画面
起動すると左下の画面になります。左クリックすると右下の画面に移ります。  
また、ESCキーでカーソルのON/OFFができます。
